## 双目操作符

### +、-、*

\+ 用来完成加法操作

\- 用来完成减法操作

\* 用来完成乘法操作

### \

\\ 用来完成除法操作

除号的两端如果是整型，则执行整型除法，得到的结果也是整型

~~~c
int x = 6;
int y = 4;
int z = x / y;//6/4=1.5
printf("%d\n",z);//1取商
//C语言中，把浮点数赋值给整型变量会丢弃小数部分
~~~

要得到含有小数的结果，必须让除号的左右操作数至少有一个是浮点数，且保存结果的变量也是浮点型。

~~~c
int x = 6;
float y = 4.0;
float z =x / y;
printf("%f\n",z);//1.500000
//printf在打印小数的时候，默认6位小数
~~~

### %

% 用来完成求模（取余）运算，即返回两个整数相除的余值。

**左右操作数必须是整型**

~~~c
//6/4=1...2
int num = 6 % 4;
printf("num=%d\n",num);//2
//取余运算的正负号与运算符的左操作数的正负号一致
printf("%d\n",11%5);//1
printf("%d\n",-11%5);//-1
printf("%d\n",-11%-5);//-1
~~~

## 单目操作符

### ++、--

自增和自减操作符，又有前置和后置之分。

口诀：前置，先自增/减，后使用；后置，先使用，后自增/减。

### +、-

正负号，EZ啦

### ()强制类型转换

~~~c
(类型)
int num = 3.14;
//3.14是double类型的数据，num是int类型的变量
//两边类型不一致，编译器会报错
int num = (int)3.14;//强制类型转换，只取整数部分
~~~
